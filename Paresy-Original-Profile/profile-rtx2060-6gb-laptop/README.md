## Profiling

This directory contains the profiling data for **Paresy**, captured on a laptop equipped with an RTX 2060 GPU and 6 GB of VRAM.

To view the profile, download and install  [NVIDIA Nsight Compute](https://developer.nvidia.com/nsight-compute) and [NVIDIA Nsight Systems](https://developer.nvidia.com/nsight-systems).
